# Privacy-Safe Snippet Sanitizer

Runnable Vue 3 baseline for the frontend task.

## What Is Included

- Vue 3 + Vite application pinned to port `8585`
- Placeholder sanitizer workspace in `src/App.vue`
- Stable DOM IDs and form controls already wired for the task flow
- Build and preview scripts for local verification

## Prerequisites

- Node.js `18` or newer
- npm

## Setup And Run

```bash
npm install
npm run dev
```

The Vite development server is pinned to `0.0.0.0:8585` and should be reachable at `http://localhost:8585`.

Additional scripts:

```bash
npm run build
npm run preview
```
